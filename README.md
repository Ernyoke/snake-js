# snake-js
The game Snake implemented with functional style in JavaScript without libraries.

# Usage
```
git clone git@github.com/Ernyoke/snake-js.git
cd snake-js
npm install
npm start
```

* Use arrow keys, wasd, or hjkl to control the snake.
* Collect apples.
* Avoid crashing.
* Ctrl-c to quit (in terminal).
